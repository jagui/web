  • <divs> have display:block by default which means they take up an entire line, nothing can share their line, they force a line break both above and below.
  • <spans> have display:inline by default which means they take up the minimum amount of space possible.
  • <img> have display:inline-block by default which means they behave like inline elements on which you can define the width and height (that's not possible for inline elements).
